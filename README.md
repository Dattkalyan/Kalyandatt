# Loss Recovery Dashboard - Excel

Interactive Excel Dashboard to analyze Profit/Loss trends across Regions and Years from 2023-2025.

### 📸 Dashboard Preview
![Loss Dashboard Preview](Loss_Dashboard_Preview.png)

### 🔥 Key Features
1. *Dynamic KPI Card:* Total Loss YTD updates automatically with Slicers using GETPIVOTDATA.
2. *Interactive Slicers:* Filter data by Region and Years to drill down into specifics.
3. *Line Chart Trend:* Visualizes monthly loss pattern to identify seasonal dips like Mar-Apr.
4. *Conditional Formatting:* Data Bars used to highlight magnitude of loss.

### 📊 Key Insights from Data
1. *Highest Loss Region:*  total loss.
2. *Seasonal Pattern:* Loss consistently peaks in *March-April* every year.
3. *Most Stable Region:* total loss.
4. *Total Loss (All Regions):* *from 2023 to June 2025.
